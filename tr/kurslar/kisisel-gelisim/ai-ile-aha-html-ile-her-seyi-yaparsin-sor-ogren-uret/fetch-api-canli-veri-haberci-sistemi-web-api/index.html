Dış Dünyadan Haber Almak: Fetch API ile Canlı Veri Çekme Sistemi

Dış Dünyaya Açılan Kapı: Fetch API

Modern web uygulamaları sadece kendi içinde çalışan sistemler değildir. İnternet üzerinden başka sistemlerle iletişim kurabilirler.

Bu iletişimin temel aracı Fetch API’dir.


https://www.udemy.com/course/ai-ile-aha-html-ile-her-seyi-yaparsn-sor-ogren-uret/


Fetch API sayesinde:

  • Dış sunuculardan veri alınır
  • JSON formatında bilgi çekilir
  • Sayfa dinamik olarak güncellenir

Fetch API Nedir?

Fetch API, tarayıcı üzerinden HTTP istekleri göndermeyi ve veri almayı sağlayan modern bir yöntemdir.

Bu sistem:

  • Sunucuya istek gönderir
  • Veriyi bekler
  • Gelen veriyi işler
  • Sayfaya yansıtır

Canlı Veri Mantığı

Canlı veri, sayfa içinde sabit olmayan, dış kaynaktan anlık gelen bilgidir.

Örneğin:

  • Günün sözü
  • Döviz kuru
  • Hava durumu bilgisi

Bu veriler sürekli değişebilir.


Haberci Sistemi Metaforu

Bu dersin metaforu “haberci” sistemidir.

  • Fetch API = haberci
  • API sunucusu = dış dünya
  • Sayfa = alıcı sistem

Haberci dışarıdan bilgi getirir ve sistemi günceller.


AI ile Veri Çekme Süreci

Yapay zekâ bu süreçte veri akışını yöneten bir yardımcıdır.

Sistem:

  • API adresini belirler
  • İsteği gönderir
  • Gelen veriyi işler
  • Ekranda gösterir

Kursiyerin Uygulama Süreci: İlk Dış Veri Bağlantısı

Öğrenci bu aşamada gerçek bir API bağlantısı kurar.

Süreç:

  • HTML dosyası hazırlanır
  • Fetch isteği eklenir
  • JSON verisi alınır
  • Ekrana yazdırılır
  • Sayfa test edilir

Veri Akış Döngüsü

Bu sistemin temel akışı şöyledir:

  • İstek gönderilir
  • Sunucu yanıt verir
  • Veri alınır
  • Sayfaya aktarılır

Bu döngü modern web uygulamalarının temelidir.


Gerçek Dünya Bağlantısı

Fetch API ile birlikte uygulama artık dış dünyaya bağlı hale gelir.

Bu sayede:

  • Gerçek veri kullanılır
  • Statik yapı kırılır
  • Dinamik içerik oluşur

Aha! Anı: Dünya ile Bağ Kurmak

Bu dersin en önemli farkındalığı şudur:

  • Sayfa artık kapalı bir sistem değildir
  • Dış veri kaynaklarına bağlanır
  • Canlı bilgi akışı sağlar

Metafor: Dijital Haberci

Bu yapı şu şekilde düşünülebilir:

  • Fetch API = haberci
  • API = şehir dışı kaynak
  • Web sayfası = merkez

Haberci sürekli yeni bilgi taşır.
